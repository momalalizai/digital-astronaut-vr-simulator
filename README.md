# Digital Astronaut – Multiplayer VR Space Mission Simulator

A Unity-based multiplayer VR training simulation designed to improve collaborative problem solving and operational readiness in space mission scenarios.

This project combines a VR astronaut role with a tablet-based mission controller, allowing real-time cooperation during maintenance tasks on a simulated International Space Station (ISS).

## Project Concept

Astronauts in space lack natural sound cues, which can reduce situational awareness and coordination during complex repair tasks.

To address this challenge, the system introduces:

- Artificial spatial audio cues
- Real-time multiplayer collaboration
- Cross-device communication
- Interactive repair missions in VR

## Core Features

### VR Astronaut Role
- Perform maintenance tasks in immersive VR
- Calibrate radar dish systems
- Align solar panels
- Navigate with robotic arm support

### Tablet Mission Controller
- Monitor astronaut actions
- Guide user through live communication
- Use drawing, highlighting, and laser tools
- Improve task coordination

### Multiplayer Networking
- Built using Photon Networking (PUN2)
- Synchronized task states
- Real-time collaboration
- Voice communication support

### Spatial Audio
- Simulated environmental sound cues
- Reinforced task outcomes
- Improved situational awareness
